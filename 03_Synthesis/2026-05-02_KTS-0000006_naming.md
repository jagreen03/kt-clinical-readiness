# Naming Recommendation: Spring Boot BFF

* Repository Name: riverhorse-gateway-spring
* Base Package: com.bluesand.riverhorse.gateway
* Rationale: Preserves the RiverHorse lineage. Reverse-DNS package structure perfectly mirrors the .NET namespace (BlueSand.RiverHorse.Gateway). Appending -spring to the repository differentiates the stack while maintaining domain grouping.
